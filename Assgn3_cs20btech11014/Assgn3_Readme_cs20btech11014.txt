To compile the program :
 
g++ Assgn3_RMS_src_cs20btech11014.cpp -o rms
g++ Assgn3_EDF_src_cs20btech11014.cpp -o edf

To run the program :

./rms
./edf

To execute this program you need an input file which contains number of processes in the first line, and the subsequent lines contain the information regarding the process i.e., process id, execution time, period, and number of times each process repeats.
